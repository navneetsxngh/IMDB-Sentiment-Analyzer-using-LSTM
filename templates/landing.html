🔄 Workflow

How It Works

Three simple steps from movie review to AI-powered sentiment insight.

1

📝 Enter Your Review

Type or paste any movie review text. You can also upload a file for batch processing — up to 50 reviews at once.

2

🧠 AI Processes It

The text is cleaned, tokenized, padded, then passed through the Bidirectional LSTM model for binary sentiment classification.

3

✅ Get Instant Results

View sentiment label, confidence percentage, probability score, and an animated visual gauge — all returned in milliseconds.

Model Evaluation

86.93% Test Accuracy
on IMDB 50K Dataset

The Bi-LSTM model was trained and evaluated on a balanced dataset of 50,000 IMDB movie reviews (25K positive, 25K negative) with an 80/20 train-test split. Callbacks include early stopping and adaptive learning rate.
